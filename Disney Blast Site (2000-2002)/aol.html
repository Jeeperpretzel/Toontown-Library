Why are certain features of Disney's Blast disabled inside the AOL embedded browser?

Two reasons:

1. The highest restriction levels within AOL block access to the Internet outside of AOL. 

As you know, Disney’s Blast is a parent-trusted, ad-free environment, but you do need to connect to the Internet to access Disney’s Blast.

2. Disney’s Blast uses standard features of Microsoft Internet Explorer that AOL does not support when that version of Internet Explorer is launched inside of AOL.

When using Disney’s Blast, the standard navigation bar on Microsoft Internet Explorer is eliminated.  This allows Disney’s Blast to be a full-screen CD-like environment that is also secure.  To gain access to this feature of Microsoft Internet Explorer, you must connect to the Internet outside of the AOL window.

For more information, please visit the Parental Controls section within AOL.
