コグバトルでギャグが選択できないまま一方的に攻撃されて負けてしまいます…。

ユーザの急激な増加に伴い、混んでいるロビーにおいてゲームサーバが誤作動を起こしてしまうことが原因です。現在、システム調整や増強の対応を進めていますが、完全改修にはしばらく時間がかかる見込みです。


対策としては、トゥーンガイドのロビーのタブにて赤信号になっているロビーの他、トゥーンタウン・セントラルが偏って混雑している様なロビーをさけてプレイしていただきますようお願い申し上げます。


※プレイ中のエリアがそれ程混んでいないように見えても、トゥーンタウン・セントラルにはたくさんのユーザがいらっしゃることがあります。その場合も障害発生の可能性が高くなることがあります。
※特に、HQ戦などの長時間に渡るバトルを開始する際は、上記に該当するようなロビーは避けてください。

 

初回インストールやアップデートの途中でダウンロードが止まっちゃいます。

お客様の環境によって、ログイン後、ゲームプレイ前のダウンロードの段階で20％で停止したり
エラーが出てプレイ画面に進めない場合があります。現在、詳しい原因を調査中ですが、
多くの場合以下の手順でプレイ画面に進む事ができます。

【操作手順】
１．一旦、トゥーンタウン・オンラインを完全に閉じてください。

２．もう一度、トゥーンタウン・オンラインのTopページ(http://toontown.disney.co.jp/)
　　またはトゥーンタウン・オンラインのデスクトップ・アイコンからゲームにログインし、
　　インストールやアップデートを再開します。
　　※トゥーンタウン・オンラインのインストール並びにゲーム・アップデートは、
　　ログイン認証後に自動で実施されます。

上記の手順でゲームが起動できるようになりますが、症状が変わらない場合は
数回この手順を繰り返してください。
 

「Windows Vista」対応について

トゥーンタウン・オンラインはWindows Vista(32bit版)に対応致しました。

2008年4月のロウボットほんぶのリリース以降今後更に利用率が増加すると見込まれるWindows Vistaに対応致しました。
尚、現行バージョンでの64bit版OSでの動作に関して問題は報告されていませんが、調査・確認ができるサンプル数が不十分と判断し推奨環境には含めていません。
予めご了承下さいます様お願いします。
 

『ひみつのともだち』とチャットをしているとゲームが強制終了しちゃう！！

『ひみつのともだち』とチャットをする際に、
　1)チャットでの発言の際にチャットウィンドウを開かない
　2)日本語入力への切り替え操作を【Alt + 半角/全角|漢字】以外で行なう
　3)漢字変換操作時に確定の為のEnterキーを使用しない
等の場合においてゲームが強制終了する問題が確認されています。

現在、この問題の解決に向けて作業を進めていますが、現状ではチャットで発言する前になるべくチャット・ウィンドウを開き、日本語入力への切り替え操作を 【Alt + 半角/全角|漢字】で行った上で、漢字への変換作業時はEnterキーで確定しながら入力をするようご協力下さい。
 

フィッシュ・ビンゴが1時間早く終わっちゃうのですが…

システム上の都合で、毎年3月第二日曜日から同11月第一日曜日まではフィッシュ・ビンゴは毎週日曜日と水曜日の午前11時から午後8時までにスケジュールが変わっています。
その他の期間は通常通り毎週日曜日と水曜日の正午から午後9時までです。
 

トゥーンタウンで遊ぶためには、どんなシステムが必要なの？

トゥーンタウンを利用するためには、Windowsのパソコンが必要です。MacintoshやNetscapeのブラウザーには対応していません。次のシステムはトゥーンタウン・オンラインが全く正常に動作するシステム構成の例ですが、条件を満たしていてもユーザ環境の設定により正常な稼動ができない場合もありますのでご了承下さい。

OS Windows XP/Vista 日本語版
※Windows 98/ME/2000での動作も可能ですが、Microsoft社の製品サポートが既に終了の為自己責任にてご利用頂きます様お願いします。
CPU Intel Pentium III(または同等クラス) 1GHz以上
メモリ 512MB以上
ハードディスク 空き容量：1GB以上
ビデオカード 3Dアクセラレーション対応のビデオカード
256MB以上のビデオメモリ
ソフトウェア Microsoft DirectX 9.0b以上
Microsoft Internet Explorer 7.0以上を推奨
 

トゥーンタウンで遊ぶためには、特別なシステムが必要なの？

トゥーンタウンのインストーラーは、あなたのパソコンに以下のソフトウエアが入っているかどうかを自動的に確認します。

Microsoft Internet Explorer 5.01以上
Microsoft DirectX 7.0以上
Macromedia Flash Player 5.0以上
 

利用する回線速度が速いほうが、ゲームで有利になるの？

トゥーンタウンは接続の速度に関係なく、アクセスした人がみんなで同時に遊べるRPGです。ただし、速度が速ければ、最初のダウンロードにかかる時間は短くなるでしょう。
 

トゥーンタウンのダウンロードは、どれくらいの時間がかかるの？

トゥーンタウンのダウンロードとインストールの所要時間は回線接続の速度によって異なります。初めてトゥーンタウンをご利用いただく場合には、約８０MBのファイルのダウンロードが必要になります。
 

1回ダウンロードすれば、何回でも遊べるの？

トゥーンタウンの内容は随時更新され、追加のダウンロードが必要になる場合もあります。ただし、更新分のダウンロードはそれほど時間はかかりません。
 

トゥーンタウンで遊んでいたら、画面が動かなくなっちゃった！

トゥーンタウンからの応答がしばらくないときは、キーボードのAltとF4を同時に押してプログラムを終了させます。終了できないときは、Control、Alt、Deleteキーを同時に押すとWindowsの「タスクマネージャ」が開くので、一覧から"toontown.exe"を選択して強制終了させます。 「タスクマネージャ」のウインドウが開かないときは、WindowsのOSがクラッシュしています。パソコンを強制終了させ、再起動してください（グラフィクスカードのドライバーに問題がある可能性が高いので、最新のドライバーを確認してください）。
 

ゲームの起動中に画面が真っ暗になったまま、元に戻らなくなっちゃった！

トゥーンタウンからの応答がしばらくないときは、キーボードのAltとF4を同時に押してプログラムを終了させます。終了できないときは、Control、Alt、Deleteキーを同時に押すとWindowsの「タスクマネージャ」が開くので、一覧から"toontown.exe"を選択して強制終了させます。 「タスクマネージャ」のウインドウが開かないときは、WindowsのOSがクラッシュしています。パソコンを強制終了させ、再起動してください（グラフィクスカードのドライバーに問題がある可能性が高いので、最新のドライバーを確認してください）。
 

ウィルス対策ソフトを利用していても、トゥーンタウンで遊ぶことはできるの？

ウィルス対策ソフトを利用していて「インターネット接続ファイアウォール」が有効になっていると、トゥーンタウンのサーバーに接続できず、ゲームのダウンロードやスタートアップ設定の最中にハングアップするする場合があります。これらの製品のファイアウォール機能は、必ず無効にしてください。
 

トゥーンタウンで遊んでいたら、Windows XPが勝手に再起動しちゃった！

Windows XPでは、OSがクラッシュすると自動的に再起動するように、デフォルト設定されています。この設定を変更するときは、Windowsの「コントロールパネル」から「システム」→「詳細設定」→「起動と回復」→「設定」を選び、必要な項目を確認してください。
　 Windows XPは、ハードウエア（メモリーチップの不具合など）かデバイスドライバーに問題が生じたとき、クラッシュする設計になっています。トゥーンタウンはビデオドライバーに大きな負担がかかるので、使用しているグラフィクスカード製品の最新ドライバーがインストールされているか、パソコンかマザーボードの製造元に確認するといいでしょう。

それでも問題が解決しない場合は、使用しているビデオカードとマザーボードの製造元のカスタマーサービスに問い合わせてください。
 

トゥーンタウンは、Windows XPとVistaしか利用できないの？

2008年4月のロウボット本部リリース以降、一般的にトゥーンタウン・オンラインはWindows 98(SE)／ME／2000／XP／Vistaの日本語版に対応していますが、Windows98／ME／2000に関しては既にMicrosoft社の製品サポートが終了している為に今後ユーザの皆さんに十分なサポートをお約束できなくなる可能性があります。その為、これらのOSの推奨を中止し自己責任によるご利用をお願いしております。尚、トゥーンタウンをインストールするときはパソコンに「管理者」のアカウントでログインする必要があります。インストール後はアカウントに関係なく利用できます。最新のToontown Launcher ActiveX controlをダウンロードするときは管理者のアカウントが必要になりますが、その場合はダウンロードを始める前に画面で通知されます。
 

トゥーンタウンは、Windowsパソコン以外のOSでも利用できる？

現在、トゥーンタウンはWindowsにのみ対応しています。それ以外のOSについては対応予定はありません。
 

トゥーンタウンで遊ぶのにピッタリの画面の解像度や色（ビット数）は？

トゥーンタウンを最初にインストールすると、あなたのパソコンに適した解像度や色（ビット数）に合わせて自動的に調節されます。インストール後は、トゥーンタウンの「オプション」から変更できます。
 

トゥーンタウンをインストールすると、パソコンのハードディスクにどんな情報が追加されるの？

トゥーンタウンでは、ゲームで遊ぶために必要なソフトウエアをあなたのパソコンのハードウエアにインストールします。そのため、ハードウエアに最低100MB以上の空き容量が必要となっています。インストールされるのはトゥーンタウンのゲームを楽しむためだけに使うソフトウエアで、パソコンのそれ以外の動作には影響を与えません。
 

トゥーンタウンをアンインストールする方法を教えて！

Windowsの「スタートメニュー」から「設定」→「コントロールパネル」→「アプリケーションの追加と削除」を選び、「 トゥーンタウン・オンライン」の「削除」を選べば、ほかのWindowsのアプリケーションと同じように削除できます。ただし、あなたがつくったトゥーンは削除されず、あなたのアカウント情報はすべてトゥーンタウンのサーバーに保存されています。再インストールしても同じアカウントで利用できます。
 

トゥーンタウンは、いろんなパソコンから利用できる？

トゥーンタウンをインストールするパソコンの台数に制限はありませんが、。あなたのアカウント情報とトゥーンはすべて、トゥーンタウンのサーバーに保存されているので、トゥーンタウンがインストールされていれば、どのパソコンからでもあなたのアカウントで利用できます。
ただし、同じアカウントで複数のパソコンから同時にログインすることは、現在のところできません。1個のアカウントにつきトゥーンを6個まで設定できるため、同じアカウントを複数の人でも使えますが、複数の人が同時にログインすることはできません。同時にログインして利用したいときは、1人ずつアカウントを取得してください。
 

ゲームの更新情報は、どうやったら教えてもらえるの？

ディズニーのテーマパークと同じように、トゥーンタウンも定期点検をして、新しいコーナーやゲームを追加していきます。更新された内容はあなたがトゥーンタウンにアクセスしたときに自動的にダウンロードされ、インストールされます。
 

トゥーンタウンにアクセスしようとしたら、「サーバーのダウン」というメッセージが出たんだけど…。

停電、落雷による回線トラブル、ゲームサーバーのトラブルなどの理由で、予告なしにサーバーを停止させ、再起動する場合があります。また、定期点検や新しいコンテンツの追加のために、サーバーを一時停止することもあります。サーバーを停止するときは、基本的にはトゥーンタウンのサイトで前もって予告します。
「サーバーのダウン」という警告が出てもあわてずに、しばらく時間をおいてから、アクセスしなおしてみましょう。警告に合わせて、再開予定の時期が表示されている場合もあります。
 

トゥーンタウンのほかのプレーヤーとコミュニケーションを取りたいときはどうすればいいの？

「ひみつのともだち」機能を利用できないメンバーも、トゥーンタウンでは「スピードチャット」を通してほかのプレーヤーにメッセージを送ることができます。スピードチャットでは、あらかじめ用意された定型の言葉や文章を選んでメッセージをつくります。
 

トゥーンタウンをインストールしようとすると、ハードディスクには十分な空き容量があるのに「メモリが足りません」というメッセージが出ちゃうんだけど？

これはハードディスクではなく、RAMメモリに関するメッセージです。32MBや64MBのメモリでは、トゥーンタウンは利用できません。最低128MB以上のメモリが必要です。
 

トゥーンタウンを利用していたら、画面がきちんと表示されなくなっちゃった！

まず、あなたが使っているグラフィクスカードの種類を確認してください。トゥーンタウンの動作保証をしていないカードではありませんか？　カードの種類に問題がなければ、ビデオカードのドライバーに問題があるかもしれません。最新のドライバをインストールしているかどうか、カードの製造元のサイトなどで確認してください。DirectXの最新バージョンは、 マイクロソフト株式会社の公式サイトからインストールできます。また、Windows 2000/XPを使っている場合は、最新のサービスパックをインストールしてください。
